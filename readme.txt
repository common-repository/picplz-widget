== Description ==

This plugin creates a Picplz widget that displays the latest photos you have posted to your Picplz account.

See it being used on my [blog](http://harrywolff.com/).

Current options include:

*   Set your Picplz username
*   [NEW] Set how many photos you want to display in the widget
*   Set the height and width of each picture displayed in the widget
*   Option to display the Picplz logo
*   Smarter include of jQuery library.  Now only included if you aren't already using the jQuery library.  Faster page loads!
*   More custom configurations coming!

Thank you to the [Picplz](http://picplz.com/ "Picplz") team for a great service, and thank you voidfiles for your work on the [Picplz API](https://sites.google.com/site/picplzapi/) and [examples](https://github.com/mxml/picplz-examples)!

== Installation ==

To install the Picplz widget:

1. Upload `picplz-widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Customize the widget through the 'Widget' menu in WordPress
1. Take more pictures!
